# 🔐 Password Strength Checker

A GUI-based cybersecurity project that checks password strength and provides suggestions.

## 🚀 Features
- Detects Weak / Medium / Strong passwords
- Suggests improvements
- Show/Hide password
- Strength meter

## 🛠️ Tech Stack
- Python
- Tkinter
- Regex
